# Tape-SATA Plug
SATA plugin board to lead the SATA FPC port on laptop motherboard out.

```
|                                         Power                                        |                SATA                 |
| -------------------------------------------------------------------------------------------------------------------------- |
| 12V | 12V | 12V | GND | GND | GND | 5V | 5G | 5V | GND | GND | GND | 3V3 | 3V3 | 3V3 | GND | B+ | B- | GND | A- | A+ | GND |
        |  |            |        |
        |  |            |        |
        |  |            |        |
BarrelJack (12V)        |  FPC   |
                        |        |
                        |        |
                        |        |
                        |        |
|      Power      |              SATA                |
| -------------------------------------------------- |
| 5V | 5V | 5V | GND | B+ | B- | GND | A- | A+ | GND |
```
